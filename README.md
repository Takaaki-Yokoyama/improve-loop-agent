# PDCA Agent

Claude Code のスラッシュコマンドを使って、AIとの対話でPDCAサイクルを回すワークスペースです。

## 使い方

### 1サイクルの流れ

```
/01-plan    → 目標・タスクを計画する
/02-do      → タスクを進める・Claudeに相談する
/03-check   → 結果を振り返る
/04-improve → 改善案をまとめてサイクルを完了する
```

### プラン番号

各サイクルには番号（ID）を付けて管理します。

- `/01-plan` 起動時に番号を確認（デフォルト: `YYYYMMDD` 形式の今日の日付）
- 全ファイルが `{プラン番号}/` フォルダにまとまるので、サイクルごとに独立して管理できる
- improve まで完了したフォルダは自動で `done/` へ移動される

### フォルダ構成

```
.current-plan        # 現在進行中のプラン番号
{プラン番号}/
  plan.md            # 目標・タスクチェックリスト
  result.md          # 振り返り結果
  improve.md         # 改善案・次サイクルへの引き継ぎ
done/
  {完了した番号}/    # improve 完了後にアーカイブされる
```

## 必要なもの

- [Claude Code](https://claude.ai/code)

## 各コマンドの詳細

| コマンド | 役割 |
|---|---|
| `/01-plan` | 前サイクルの引き継ぎを確認しながら今サイクルの目標・タスクを決める |
| `/02-do` | plan.md を参照しながら作業を進める。タスク完了をチェックボックスに反映 |
| `/03-check` | 達成率を分析し、うまくいったこと・課題を result.md にまとめる |
| `/04-improve` | 改善案を引き出して improve.md にまとめ、サイクルを done/ へアーカイブ |
