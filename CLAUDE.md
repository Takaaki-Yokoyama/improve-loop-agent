# PDCA Agent

このプロジェクトはPDCAサイクルをAIとの対話で回すためのワークスペースです。

## ワークフロー

```
/01-plan    → プラン番号を確認し {番号}/plan.md 作成（目標・タスク計画）
/02-do      → {番号}/plan.md を元に作業進行・壁打ち
/03-check   → 結果を {番号}/result.md にまとめる
/04-improve → 改善案を {番号}/improve.md にまとめる（次サイクルへ）
```

## プラン番号

- `/01-plan` 起動時にユーザーに確認する（デフォルト: 今日の日付を `YYYYMMDD` 形式にしたもの）
- 確定したプラン番号は `.current-plan` ファイルに保存され、後続フェーズで参照される
- 各サイクルのファイルはすべて `{プラン番号}/` フォルダ内に保存される

## ファイルの役割

| ファイル | 内容 |
|---|---|
| `.current-plan` | 現在アクティブなプラン番号（サイクル完了時に削除） |
| `{番号}/plan.md` | 現サイクルの目標・タスクチェックリスト |
| `{番号}/result.md` | 現サイクルの結果まとめ |
| `{番号}/improve.md` | 改善案・次サイクルへの引き継ぎ |
| `done/{番号}/` | improveまで完了したサイクルのアーカイブ |

## Claudeへの基本姿勢

- ユーザーが作業中に質問してきたら `{番号}/plan.md` のコンテキストを踏まえて答える
- タスクの完了/未完了は `{番号}/plan.md` のチェックボックスに反映する
- 各フェーズのファイル出力は日本語で行う
